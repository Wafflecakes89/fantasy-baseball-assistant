# Fantasy Baseball AI Draft Assistant (Expanded)

This version includes:
- 340-player seeded pool
- Live rankings
- AI recommendation panel
- My Team tracking
- Category totals
- CSV import
- Tiered cheat sheet board by position
- Editable league teams and draft slots
- Full draft history with round and pick-in-round tracking

## Deploy on Vercel
1. Unzip this folder.
2. Upload the contents to a GitHub repository.
3. In Vercel, import the repository.
4. Make sure:
   - Framework Preset = Next.js
   - Root Directory = the folder containing package.json
   - Output Directory = blank
5. Deploy.
